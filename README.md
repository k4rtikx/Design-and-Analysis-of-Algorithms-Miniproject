# Train Scheduling using Graph Algorithms

## Overview
This project models a railway network as a graph and optimizes train movement.

- Stations → Nodes  
- Tracks → Edges  
- Travel time → Weights  

## Algorithms Used
- Dijkstra Algorithm → finds shortest path  
- Greedy Scheduling → resolves conflicts  

## Approach
- Find shortest path for each train  
- Simulate train movement  
- If conflict occurs:
  - Shorter remaining path gets priority  
  - If equal → First Come First Serve
